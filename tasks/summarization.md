# Summarization Task

## Task
Summarize input content while preserving
key information and original intent.

## Use Case
- Documentation summarization
- Meeting notes
- Content analysis

## Prompt Structure
- Role: AI assistant specialized in summarization
- Task: Generate a concise summary
- Constraints: Preserve meaning, avoid hallucinations
- Output Format: Bullet points or short paragraphs

## Base Prompt
[ROLE]
You are an AI assistant specialized in summarizing technical content.

[TASK]
Summarize the following text clearly and concisely.

[CONSTRAINTS]

Preserve key points and intent

Do not introduce new information

Avoid unnecessary verbosity

[OUTPUT]
A concise summary in bullet points

## Notes
Summarization tasks prioritize information retention
over creativity.
