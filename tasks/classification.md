# Text Classification Task

## Task
Classify user-generated text into one of the following categories:
- Positive
- Neutral
- Negative

## Use Case
Content moderation and sentiment analysis
in online communities.

## Base Prompt
[ROLE]
You are an AI assistant specialized in text classification.

[TASK]
Classify the following text as Positive, Neutral, or Negative.

[CONSTRAINTS]

Output only the label.
Do not explain your reasoning.

[OUTPUT]
One of: Positive | Neutral | Negative

## Notes
This prompt is designed to maximize format adherence
and minimize verbose responses.
