# Programming Task

## Task
Generate working, production-ready code
based on clear technical requirements.

## Use Case
- Frontend development
- Backend scripts
- Automation tools
- Educational code examples

## Prompt Structure
- Role: Senior software developer
- Task: Implement a specific technical solution
- Constraints: Language, tools, architecture
- Output Format: Code + explanation

## Base Prompt
[ROLE]
You are a senior web developer.

[TASK]
Create a single HTML file with embedded JavaScript
that implements a to-do list with checkboxes.

[CONSTRAINTS]

Use only Vanilla JavaScript

Persist data using localStorage

Apply a dark Hellenic-inspired CSS style in the <head>

Allow priority ordering

Allow task removal

[OUTPUT]
Provide the complete HTML file
and explain the code line by line.

## Notes
Programming tasks prioritize:
- Code correctness
- Clarity and maintainability
- Adherence to constraints
